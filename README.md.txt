Movie Recommendation System using NLP

This project is a content-based movie recommendation system that suggests movies based on storyline similarity using NLP techniques.

#Technologies Used
- Python
- Pandas
- NLTK
- Scikit-learn
- Streamlit

# Features
- Storyline preprocessing
- TF-IDF vectorization
- Cosine similarity-based recommendations
- Interactive Streamlit UI

# How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
